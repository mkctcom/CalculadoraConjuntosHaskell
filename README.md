# CalculadoraConjuntosHaskell

Descrição:

Calculadora de operações de conjuntos matemáticos utilizando o paradigma Funcional. O projeto foi escrito utilizando a linguagem de programação Haskell e a biblioteca GTK3 para a interface gráfica.

Nos campos “Conjunto A” e “Conjunto B”, o usuário insere os conjuntos a serem calculados, separando os elementos por vírgula. Em seguida, o usuário clica no botão referente a operação desejado, feito isto o resultado será mostrado no campo “Saída”.

Requisitos Funcionais:

1. Calcula União, Intersecção e Diferença entre dois conjuntos dados como entrada.
2. Verifica se um primeiro conjunto é Subconjunto e/ou Subconjunto Próprio de um
segundo.
3. Aceita conjuntos vazios.
4. Aceita conjuntos de números inteiros separados por virgula.
5. Aceita conjuntos de caracteres separados por virgula.
6. Uma tela de interaface de usuário.

Requisitos Não-Funcionais:

1. Aplicação Desktop.
2. O sistema foi implementado na linguagem Haskell.
3. É executável nas plataformas Windows e Linux/Ubuntu.
